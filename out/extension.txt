### CreateLabel
A simple label control supports custom text size,
spacing (Top/Center/Bottom Padding), background color, and direction, etc.

| Parameter | Type
| - | - |
| arrangement | component
| text | text
| textSize | number
| textStyle | LabelViewStyle <small><mark>(helper blocks)</mark></small>
| textColor | number
| number | text
| numberStyle | LabelViewStyle <small><mark>(helper blocks)</mark></small>
| numberSize | number
| numberColor | number
| direction | LabelViewDirection <small><mark>(helper blocks)</mark></small>
| topPadding | number
| centerPadding | number
| bottomPadding | number
| backgroundColor | number

* Enums for **LabelViewDirection**: `TopLeft`, `TopRight`
